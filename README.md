# Pomodoro Desktop (Этап 1)

Desktop-приложение на `PySide6` с единым состоянием таймера и двумя окнами:
- главное окно
- плавающее окно

## Требования
- Python 3.11+
- Windows/macOS/Linux

## Установка
```bash
python -m venv .venv
.venv\Scripts\activate  # Windows
pip install -r requirements.txt
```

## Запуск
```bash
python main.py
```

## Настройки
Файл `settings.json` создается автоматически в корне проекта при первом запуске.

Поля:
- `pomodoro_minutes`
- `short_break_minutes`
- `long_break_minutes`
- `long_break_interval`
- `always_on_top_default`

## Что реализовано
- Sidebar: `Pomodoro Time`, `Планирование`, `Настройки`
- Режимы: `Помодоро` / `Короткий перерыв` / `Длинный перерыв`
- Стрелки `▲/▼` рядом с таймером для изменения длительности текущего режима
- Большая SVG-иконка часов под цифрами таймера
- Плавающее окно в классическом компактном виде с pin-кнопкой (SVG)
- При pin:
  - окно поверх всех окон
  - позиция окна фиксируется
- Логика циклов без автозапуска:
  - после завершения сессии таймер переключается на следующий режим
  - следующий режим запускается вручную кнопкой `СТАРТ`
- Надежный звуковой сигнал по окончании сессии (Windows + fallback)
- Полная синхронизация главного и плавающего окон

## Архитектура
- `app/core` - состояние, контроллер, настройки
- `app/ui` - главное окно, плавающее окно, диалог настроек, стили
- `app/utils` - утилиты
- `app/assets` - SVG-ассеты
